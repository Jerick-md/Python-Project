#### "Gemini Ai Chatbot"

The project is a simple command-line chatbot built using Python and Google's Gemini

Generative AI (via the google-generative library). It allows users to chat with an AI model

in real time until they choose to exit.



#### Features

&nbsp;-Interactive command-line chat

&nbsp;-Uses Gemini 2.0 Flash model

&nbsp;-Continuous conversation using chat history

&nbsp;-Simple exit command



#### Requirements

&nbsp;-Pyhton 3.0 or higher

&nbsp;-google-generativeai library

&nbsp;-A valid Google Gemini API key



#### Installation

1. Install Python from https://www.python.org
2. Install the required library:

&nbsp;   "pip install google-generativeai"

#### 

#### Setup

Open the Python file and replace the API key with your own Gemini API key:

&nbsp;

&nbsp; "genai.configure(api\_key='YOUR\_API\_KEY')"



#### Usage

Open Vscode or any IDE,

Run the script using the command below:

&nbsp;  "python geminai.py"



Type your message and press Enter to chat with the AI.

Type 'x' and press Enter to exit the program
